installa:

ricorda di essere nella directory che ti interessa

ricorda di usare => npm init -y => crea il file package.json

-babel-cli => npm install --save-dev babel-cli
-babel-preset-env => npm install --save-dev babel-preset-env
-babel-preset-es2015 => npm install --save-dev babel-preset-es2015 //Questa non è necessario se si installa "babel-preset-env" in quanto env funzione come se i usase -es2015 -es2016 -es2017 tutti insieme
-babel-preset-react => npm install --save-dev babel-preset-react
-babelify => npm install --save-dev babelify
-browserify => npm install -g browserify
-react => npm install react
-react-dom => npm install react-dom
-watchify => npm install -g watchify
-sass nodemon => npm install -D node-sass nodemon
-redux => (npm install --save redux) (npm install --save react-redux) e (npm install --save-dev redux-devtools)

da react 15.5 +
-prop-types => npm install --save prop-types


ATTENZIONE: ricorda di controllare il file package.json e controlla che siano presenti tutte le dipendenze dei moduli installati
